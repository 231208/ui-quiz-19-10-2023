Q1 : What is console.log()

Console.log() method outputs a message to the web console. The message may be a single string (With optional substitution values), or it may be any more Javascript objects.


syntax
let a = 10;
let b = 12;
c = a + b;
console.log(c)

Q3 : Where we debug our web page Js code ?

Js code debuging Debugging is not easy. But fortunately, all modern browsers have a built-in javascript debugger.
The debugger Keyword
The debugger keyword stops the execution of Javascript, and calls(if available) the Debugging function.
This has the same function as setting a breakpoint in the debugger.
If no Debugging is available, the debugger statement has no effect.
